## Laravel Vue Chatapp

Realtime Chatapp using Laravel, VueJS, and Pusher.

- git clone https://github.com/nukipratama/laravel-vue-chatapp.git
- cd laravel-vue-chatapp/
- composer install
- cp .env.example .env
- Edit .env
- php artisan key:generate
- npm install && npm run dev
- php artisan serve
- enjoy:)
